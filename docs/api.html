Quantum Fortuna API Documentation

This documentation covers the main modules and functions available in the Quantum Fortuna educational project.

Random Number Generator (RNG) Module

quantum_rng(min_val=1, max_val=49, count=6)

Generates quantum-inspired random numbers using simulated quantum noise.

Parameters:

min_val (int): Minimum value for the random numbers (inclusive)

max_val (int): Maximum value for the random numbers (inclusive)

count (int): Number of random numbers to generate

Returns:

List[int]: A list of unique random integers

Example:

from quantum_fortuna.rng import quantum_rng

# Generate 7 random numbers between 1 and 49
numbers = quantum_rng(min_val=1, max_val=49, count=7)
print(numbers)  # Example output: [23, 7, 42, 15, 31, 9, 38]

classical_rng(min_val=1, max_val=49, count=6)

Generates random numbers using Python's random module.

Parameters:

min_val (int): Minimum value for the random numbers (inclusive)

max_val (int): Maximum value for the random numbers (inclusive)

count (int): Number of random numbers to generate

Returns:

List[int]: A list of unique random integers

Modeling Module

RandomBaselineModel()

A baseline model that generates random predictions.

Methods:

fit(X, y): Fits the model to training data (does nothing in this case)

predict(X): Makes random predictions

Example:

from quantum_fortuna.modeling import RandomBaselineModel

model = RandomBaselineModel()
model.fit(None, None)  # No actual training happens
predictions = model.predict(10)  # Generate 10 random predictions
print(predictions)

Pipeline Module

process_historical_data(file_path)

Processes historical lottery data from a CSV file.

Parameters:

file_path (str): Path to the CSV file containing historical lottery data

Returns:

pandas.DataFrame: Processed data ready for analysis

evaluate_prediction(prediction, actual)

Evaluates the accuracy of a lottery prediction.

Parameters:

prediction (List[int]): Predicted lottery numbers

actual (List[int]): Actual lottery numbers

Returns:

dict: Evaluation metrics including match count and accuracy

Example:

from quantum_fortuna.pipeline import evaluate_prediction

prediction = [1, 7, 13, 25, 31, 42]
actual = [7, 13, 22, 31, 35, 42]

result = evaluate_prediction(prediction, actual)
print(f"Matched {result['match_count']} numbers")
